Objective :

The objective of this lab is to introduce students to the basics of HTML (Hypertext Markup Language) and provide hands-on experience in creating simple web pages.

Introduction :

HTML is the standard markup language used to create web pages. It provides a structure for content on the World Wide Web by using a system of tags and attributes to define elements such as headings, paragraphs, links, and images.This lab aims to introduce students to HTML and guide them in creating their own simple website. Understanding HTML is essential for anyone interested in web development, as it forms the backbone of every webpage.

Materials:

  • Text editor (Sublime Text, Visual Studio Code)
  • Web browser (e.g., Google Chrome, Mozilla Firefox, Safari)
  • Internet connection(for accessing additional resources if needed)

Conclusion :

In conclusion, this lab was provided a foundational understanding of HTML, its syntax, and its role in creating web pages. Through hands-on practice students had learned how to structure a webpage, add content, and utilize essential HTML elements. We are now equipped with the basic skills to begin creating our own web pages.By continuing to practice and explore HTML, students can build upon their skills and create more complex and dynamic websites in the future.
